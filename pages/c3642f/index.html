win10安装LxRunOffline 管理wsl

# 安装原因

  • php开发用到了swoole，windows不行
  • worker gateway windows不行
  • 嗯 不行 但是qq 微信 软件不错

# 第一步 开启wsl

  • 复制 Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Windows-Subsystem-Linux
  • 打开 powershell 管理员，粘贴，回车执行，后面需要重启电脑 yes即可

假如已经开启过windows子系统的(wsl)可以省略此步骤。

# 第二步 安装包管理器

  • 前置 安装一个安装LxRunOffline的工具

  • 有两个选择 choco 和 scoop

  • choco(推荐)

# 复制下面一行的命令
Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))

# 第三步 安装LxRunOffline

# choco方式 
choco install lxrunoffline

# scoop方式
scoop bucket add extras
scoop install lxrunoffline

# 第四步 下载系统镜像

打开网址 https://github.com/CentOS/sig-cloud-instance-images/blob/CentOS-7.8.2003-x86_64/docker/centos-7.8.2003-x86_64-docker.tar.xz文件

这个是centos的镜像

更多其他镜像可参考:https://www.cnblogs.com/weihua2020/p/13926271.html

# 第五步 安装镜像

LxRunOffline install -n [自定义名称,例如:centos] -d [安装位置] -f [刚才下载镜像文件的位置] -s


# LxRunOffline install -n CentOS -d D:\WSL\centos -f D:\WSL\IOS\centos-7.8.2003-x86_64-docker.tar.xz -s

# 第六步 成功

  • 可以鼠标右键，选择LxRunOffline,然后选择centos打开命令，进入linux系统命令
  • 直接在cmd 或者 powershell 输入wsl 执行，进入linux系统命令

两种方法都可
