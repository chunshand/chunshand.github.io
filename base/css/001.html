原理 #

  • nav nav-mask分别使用 sticky 定位，固定在了top:0 、top:48 的位置
  • nav 的z-index 要大于 nav-shaow
  • nav-mask 通过决定定位，默认可遮挡 nav-shaow 元素
  • 滚动后nav-mask会随着list整体向上,nav-shaow则会显示出来
  • 设置nav-shaow为阴影样式
